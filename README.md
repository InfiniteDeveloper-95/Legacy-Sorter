# Legacy Sorter (Alpha)

Утилита для организации файлов и работы с игровыми архивами. 
Дизайн выполнен в классическом стиле Windows 95/98.

### 🔍 Основные функции:

* **Сортировка:** Быстрое распределение файлов из конкретной папки и подпапок в одну папку.
* **Архиватор:** Поддержка распаковки форматов `.wad`, `.pak` и специфических `.apf`, `.brarchive` и другие.
* **Работа с файлами:** Валидация и форматирование JSON и другое.
* **Конвертирование:** конвертация одного формата файла в другой.

### 🚀 Как запустить:
1. Перейдите в раздел **Releases** на странице проекта.
2. Скачайте архив с последней версией.
3. Запустите `LegacySorter.exe`.
